Impostazioni
1

Autorizza YouTube (una volta sola, dal tuo PC)

Questo genera un token OAuth che GitHub Actions usa per scaricare senza bot-detection. Fatto una volta, dura a lungo.

A — Installa e autorizza (terminale Windows)

pip install yt-dlp
pip install https://github.com/coletdjnz/yt-dlp-youtube-oauth2/archive/refs/heads/master.zip
yt-dlp --username oauth2 --password "" https://www.youtube.com/watch?v=dQw4w9WgXcQ

Apri il link che appare → accedi con Google → "Consenti" → torna al terminale e attendi.

B — Copia il token negli appunti (PowerShell)

$f="$env:APPDATA\yt-dlp\youtube-oauth2\token_data.json"
[Convert]::ToBase64String([IO.File]::ReadAllBytes($f)) | Set-Clipboard

C — Aggiungi come Secret su GitHub

github.com → tuo repo → Settings → Secrets and variables → Actions → New repository secret

  • Nome: YT_OAUTH_TOKEN
  • Valore: incolla dagli appunti (stringa base64 lunga)
Il token OAuth YouTube è valido a lungo — raramente serve rinnovarlo.
2

Crea il bucket Supabase Storage (una volta sola)

Su supabase.com → progetto → Storage → New Bucket:

  • Nome: ytp-audio
  • Abilita Public bucket ✓
  • Clicca Create
3

Token GitHub per avviare i download

Genera un Personal Access Token con scope workflow:

github.com → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token → spunta workflow

Clicchi ⬇ nell'app → GitHub Actions scarica via yt-dlp + OAuth YouTube → MP3 su Supabase → icona 🎵. Nessun PC necessario.
4

Alternativa — Script locale sul tuo PC

Se preferisci usare il PC come downloader: scarica ytp-downloader.py + ytp-config.py da GitHub, inserisci la service_role key in ytp-config.py e avvia con doppio clic su ytp-downloader.bat.

pip install pytubefix websockets yt-dlp

Lo script resta in ascolto su Supabase Realtime e scarica automaticamente i video in coda.
