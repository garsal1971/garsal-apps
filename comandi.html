Ultime esecuzioni — fill-notification-queue
Mostra le ultime 10 esecuzioni del cron fill-notification-queue con stato e dettagli.
SQL PROD cron.job_run_details 
SELECT runid, start_time, end_time, status, return_message
FROM cron.job_run_details
WHERE jobname = 'fill-notification-queue'
ORDER BY start_time DESC
LIMIT 10;

Stato coda notifiche (cm_notification_queue)
Conta le notifiche per stato — utile per verificare che fill-notification-queue stia popolando la coda.
SQL PROD cm_notification_queue 
SELECT status, COUNT(*) AS totale
FROM cm_notification_queue
GROUP BY status
ORDER BY status;

Prossime notifiche pending
Le prossime 20 notifiche in stato pending ordinate per fire_at.
SQL PROD cm_notification_queue 
SELECT id, rule_id, app, title, body, fire_at, occurrence_id
FROM cm_notification_queue
WHERE status = 'pending'
ORDER BY fire_at ASC
LIMIT 20;

Vault — verifica chiave supabase_service_role_key
Controlla che il Vault contenga la chiave usata dal cron per autenticarsi alle edge functions. Se non esiste, il cron riceve 401 e non esegue.
SQL PROD vault.decrypted_secrets 
SELECT name, created_at, updated_at
FROM vault.decrypted_secrets
WHERE name = 'supabase_service_role_key';
