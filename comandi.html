Una sola volta GitHub
1
Crea un GitHub Personal Access Token:
github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token
Permesso necessario: Actions → Read and write.
2
Apri il Test Dashboard (link sotto), incolla il PAT nel campo "GitHub Personal Access Token" e premi Salva Token GitHub.
